# 🏠 Mortgage Amortization Calculator Pro

A high-precision web application designed to calculate and visualize mortgage payments.

### 🚀 Key Features
* **Dynamic Amortization**: Real-time calculation of principal vs. interest.
* **Data Visualization**: Integrated Chart.js for interactive financial trends.
* **Professional Export**: Custom-built PDF reporting tool using html2pdf.js with optimized layout rendering.

### 🛠️ Tech Stack
* JavaScript (ES6+), HTML5, CSS3, Chart.js, html2pdf.js.
